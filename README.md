# Aura Music - Modern Music Player for Android

Aura Music is a powerful, feature-rich music player application built with Kotlin for Android devices. Enjoy your favorite songs with an elegant interface and seamless audio experience.

## Features

- 🎵 **High-Quality Audio Playback** - Crystal clear sound with support for multiple audio formats
- 🎨 **Beautiful User Interface** - Modern, intuitive design optimized for music lovers
- 📱 **Android Optimized** - Fully native Kotlin implementation for superior performance
- 🔊 **Advanced Controls** - Full playback control including shuffle, repeat, and queue management
- 🎯 **Responsive Design** - Smooth animations and fast navigation

## Getting Started

### Prerequisites

- [Android Studio](https://developer.android.com/studio) (Latest version recommended)
- Android SDK 21 or higher
- Java Development Kit (JDK)

### Installation & Setup

1. **Clone or Download the Repository**
   - Open Android Studio
   - Select **Open** and choose the directory containing this project

2. **Import Project**
   - Allow Android Studio to automatically fix any dependencies and incompatibilities
   - Wait for Gradle sync to complete

3. **Configure API Keys**
   - Create a file named `.env` in the project root directory
   - Add your API configuration keys (refer to `.env.example` for the template)
   - Set `GEMINI_API_KEY` with your API credentials

4. **Build Configuration**
   - Open the app's `build.gradle.kts` file
   - Remove the line: `signingConfig = signingConfigs.getByName("debugConfig")`

5. **Run the Application**
   - Connect an Android device or launch an emulator
   - Click **Run** in Android Studio to build and deploy the app
